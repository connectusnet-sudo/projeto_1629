# Projeto: Análise do Mercado de Tecnologia no Brasil---
**Objetivo:** Analisar os dados do Stack Overflow Survey para entender o perfil profissional brasileiro.
**Dataset:** `survey_results_public.csv` (Filtrado para `Country == 'Brazil'`)
**Ferramentas:** Pandas para manipulação e Matplotlib para visualização.

📊 Análise de Dados: Ecossistema de Programação no Brasil
Disciplina: Técnicas de Programação

Tecnologias: Python, Pandas, Matplotlib

📝 Descrição
O projeto consiste no tratamento e análise exploratória de dados provenientes da pesquisa anual do Stack Overflow. A análise foi segmentada para o público brasileiro, permitindo uma visão detalhada sobre:

Limpeza de Dados: Tratamento de valores nulos, remoção de outliers salariais e conversão de tipos de variáveis.

Análise Comparativa: Diferença salarial entre usuários de diferentes linguagens (ênfase em Python e C#).

Demografia Profissional: Distribuição de modelos de trabalho (Remoto/Híbrido) e sistemas operacionais mais utilizados no país.
